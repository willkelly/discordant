# discordant

A vibecoded XMPP based chat client with support for video / audio streaming.  It is NOT discord.

## Technology

deno
typescript 
svelte + capacitor

## Testing

* deno test

* playwright tests

* Tested against prosody

## Features

* Native integration where appropriate (webcam, camera)
  * Android
  * Iphone
  * Web
  * Windows
  * Linux
  * OSX

* file uploads

* inline image rendering

* Sharable client side scripting
